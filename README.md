# Copilot CLI Intro Session

Internal hands-on workshop for non-coders (PTS-level). 60 minutes, hybrid delivery.

## What's in here

```
deck/                  Presentation files
  cli-intro.html         Main slide deck (open in browser, arrow keys to navigate)
  cli-intro-presenter.html  Live presenter notes (press P from deck to open)
  cli-intro-storyboard.html Full storyboard with speaker script + demo plans
pre-work/              Send to attendees before the session
  setup-guide.md         Step-by-step install + config
  verify-checklist.md    5-point ready check
demos/                 Presenter demo scripts
  demo-01-basics.md      Calendar, email, PMX, screenshots
  demo-02-pmx-connect-prep.md  Full Connect prep scenario
  demo-03-power-moves.md /research + PMX/MSX → partner roadmap
exercises/             Attendee exercise sheets
  exercise-01-first-conversation.md  Guided: calendar, email, PMX
  exercise-02-pmx-analysis.md        Guided + self-paced challenges
  exercise-03-scenarios.md           5 self-paced partner scenarios
reference/             Keep-after-session materials
  cheat-sheet.md         Commands, prompts, tips
  troubleshooting.md     Common issues + fixes
```

## Quick start

1. Send `pre-work/` to attendees ahead of time
2. Open `deck/cli-intro.html` in a browser to present
3. Press **P** during the deck to open presenter notes
4. Open `deck/cli-intro-storyboard.html` for full prep view with speaker script

## Prerequisites

- Copilot CLI installed (`winget install GitHub.Copilot`)
- GitHub login with active Copilot subscription
- Azure CLI logged in (`az login --tenant 72f988bf-...`)
- MCP servers configured: PMX, GitHub, M365
