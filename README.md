# Copilot CLI Intro Session

> A hands-on intro to GitHub Copilot CLI for partner-facing roles at Microsoft.  
> **No coding experience needed.**

## 🆕 New here? Start with [GETTING STARTED](v2/GETTING-STARTED.md)

It explains what a repo is, how to find the exercise prompts, and translates all the jargon.

---

## What's in here

The **`v2/`** folder has the current session materials:

| Folder | What's inside | Who needs it |
|--------|-------------|-------------|
| `v2/exercises/` | **Exercise prompts** — copy-paste these into Copilot CLI | Everyone (during session) |
| `v2/reference/` | Cheat sheet, troubleshooting, resource links | Everyone (during + after) |
| `v2/pre-work/` | Setup guide + checklist | Everyone (before session) |
| `v2/deck/` | Slide deck + storyboard | Facilitator |
| `v2/demos/` | Demo scripts with talking points | Facilitator |
| `v2/extras/` | Optional add-ons (overview dashboard, etc.) | Anyone who wants more |
