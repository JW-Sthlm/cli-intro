# Exercise 3 — Pick Your Scenario

> **Duration:** ~7 min  
> **Format:** Self-paced — pick one scenario and work through it  
> **Goal:** Apply what you've learned to a real partner work task

---

Pick the scenario that's most relevant to your work right now. Each one builds on the tools you've already used.

---

## Scenario A: Fix My Outcome Gaps

*"I know I have projects without outcomes. Help me find and fix them."*

**Step 1:**

```
List my projects that are missing outcomes. For each, show the partner name and project type.
```

**Step 2:** For each gap, search for matching opportunities:

```
Search MSX opportunities for [partner name] in EMEA. Show open ones with deal size.
```

**Step 3:** Link them:

```
Link the opportunity "[opp name]" to project "[project name]"
```

**Bonus:** Ask Copilot to summarize what you fixed:

```
Give me a summary of the outcome links I just created — partner, project, and opportunity for each.
```

## Scenario B: Prepare for a Partner Connect

*"I have a Connect meeting with a partner. Help me prepare."*

**Step 1:**

```
Show me all projects and deliverables for [partner name]. Include status and due dates.
```

**Step 2:**

```
Search MSX for open opportunities with [partner name]. Any that are stalled or need attention?
```

**Step 3:**

```
Based on what you found, draft 4 talking points for my Connect meeting. Focus on project health, opportunity pipeline, and any risks.
```

**Step 4:** (Optional) Draft an email:

```
Draft a short prep email to the PDM for [partner name] with these talking points.
```

## Scenario C: Draft a Partner Email Using Context

*"I need to write a follow-up email and I want it informed by actual data."*

**Step 1:**

```
Show me my recent projects with [partner name] and their current status.
```

**Step 2:**

```
Search my recent emails from or to people at [partner name]. What were the last topics discussed?
```

**Step 3:**

```
Draft a follow-up email to [contact name] at [partner name]. Reference our active projects and the last email thread. Tone: professional but warm.
```

---

## Scenario D: Analyze a Screenshot

*"I have something on my screen I want Copilot to help with."*

**Step 1:** Take a screenshot of something relevant:
- A PMX dashboard
- A partner email
- A Teams chat
- An error message

**Step 2:**

```
Look at my last screenshot. What do you see? Summarize the key information.
```

**Step 3:** Ask a follow-up:

```
Based on that screenshot, what action should I take next?
```

or

```
Draft a response to the message in that screenshot.
```

## Scenario E: Research + Roadmap

*"I want to combine market research with our pipeline data to suggest a partner roadmap."*

**Step 1:**

```
/research What are the top AI workloads that ISV partners are building on Azure in 2025-2026? Focus on Nordic/EMEA market.
```

**Step 2:**

```
Now search MSX opportunities in EMEA related to AI or machine learning. Show me the top 10 by deal size.
```

**Step 3:**

```
Based on the research and opportunity data, draft a suggested technical roadmap for a partner looking to build AI solutions on Azure. Include: recommended Azure services, skills needed, and 3 concrete project ideas.
```
